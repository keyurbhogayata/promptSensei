# Task Completion Workflow
When a task is completed, ensure the following:
1. **Testing:** Run `npm test` to ensure all tests pass.
2. **Build:** Run `npm run build` to verify the code compiles correctly.
3. **Graph Update:** Run `graphify update .` to keep the knowledge graph synchronized.
4. **Documentation:** Update `README.md`, `GEMINI.md`, or `FUTURE_IDEAS.md` if necessary.
5. **Commit:** Commit changes with an imperative message and appropriate scope.
