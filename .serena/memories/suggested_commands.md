# Suggested Commands
## Development
- `npm install`: Install dependencies.
- `npm run build`: Compile TypeScript.
- `npm test`: Run all tests.
- `npm run start:mcp`: Start the MCP server.

## Running the Application
- `node dist/cli.js review <path-to-chat-log.md>`: Review a chat log via CLI.

## Publishing
- `npm publish`: Publish the core package.
- `cd vscode-extension; npx @vscode/vsce package`: Package the VS Code extension.

## Tools
- `graphify update .`: Synchronize the knowledge graph.
